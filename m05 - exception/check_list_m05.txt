ex00

Существует Makeifle, который компилируется с использованием соответствующих 
lfags.

Есть класс Бюрократа.

У него постоянное имя.

Его оценка варьируется от 1 (высший) до 150 (низший).

Исключения возникают при попытке создать бюрократа со слишком 
высокой / низкой оценкой.

Есть геттеры для атрибутов.

Есть функции для увеличения / уменьшения оценки, 
они при необходимости выдают исключения.

Помните, что повышение оценки до 3 дает вам оценку 2, 
поскольку 1 - это наивысшая оценка ...

Используемые исключения наследуются от std :: exception или 
от чего-то производного от std :: exception 
(т.е. они могут быть уловлены как std :: exception & e). 

Существует оператор << для перегрузки потока, 
который выводит информацию о Бюрократе.



ex01

Существует Makeifle, который компилируется с использованием соответствующих 
lfags.

Есть класс Form.
У него есть имя, логическое значение, указывающее, подписан ли он (вначале это не так), 
оценка, необходимая для его подписания, и 
оценка, необходимая для его выполнения.

Имя и сорта неизменны.
Все эти атрибуты являются частными и не защищены. 
Оценки имеют те же ограничения, что и у Бюрократа (исключения, 1 = высший, 150 = низший и т. Д.).
Существуют геттеры для атрибутов и 
оператор << для перегрузки потока, 
который отображает полное состояние формы. 
Существует функция-член Form :: beSigned, которая работает, 
как описано субъектом. 
Существует функция Bureaucrat :: signForm, которая работает так, как описано субъектом.



ex02

Существует Makeifle, который компилируется с использованием соответствующих 
lfags. 
Существуют конкретные формы, соответствующие спецификациям предмета 
(обязательные оценки, имена и действия). 
В конструкторе они принимают только один параметр - цель. 
Существует метод Form :: execute (Bureaucrat const & executor), 
который работает так, как определено субъектом. 
Либо этот метод является чистым, и проверки оценок реализованы в каждом подклассе, 
либо этот метод выполняет проверки, а затем вызывает другой метод, 
который только выполняет действие и является чистым в базовом классе, 
оба этих метода допустимы. 

Существует Bureaucrat :: executeForm (Form const & form), который работает так, как определено субъектом.
